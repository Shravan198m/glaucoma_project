# Consolidated Glaucoma Project Notebook

This directory contains a consolidated Jupyter notebook that combines the content of six sub-notebooks:

- 01_preprocessing.ipynb
- 02_segmentation.ipynb
- 03_cdr_calculation.ipynb
- 04_training.ipynb
- 05_evaluation.ipynb
- 06_final_output.ipynb

## Files

- `glaucoma_project_consolidated.ipynb`: The consolidated notebook
- `consolidate_notebooks.py`: Script to generate/update the consolidated notebook
- `README_CONSOLIDATED.md`: This file

## How it works

The consolidated notebook is generated by running `consolidate_notebooks.py`. This script:
1. Reads each sub-notebook in order
2. Strips outputs from all code cells (to reduce file size and avoid redundancy)
3. Combines all cells into a single notebook
4. Adds an introductory markdown cell explaining the consolidation

## Updating the consolidated notebook

After making changes to any of the sub-notebooks, run the following command in this directory:

```bash
python consolidate_notebooks.py
```

This will overwrite `glaucoma_project_consolidated.ipynb` with the updated content.

## Notes

- The stripped outputs mean that the consolidated notebook is lightweight and can be version-controlled efficiently.
- To see outputs, run the notebook cells after opening the consolidated notebook.
- The consolidation preserves the original order of cells from each sub-notebook.
- Markdown cells from the sub-notebooks are included (except for outputs in code cells).

## Requirements

- Python 3.x
- No additional packages required (uses only standard library)
